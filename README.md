# زياد للطباعة — Ziad Printing

موقع شركة زياد للطباعة مع لوحة مطور متصلة بسيرفر Supabase.

## خطوات التشغيل

### 1) إعداد السيرفر (مرة واحدة)
1. ادخل إلى مشروعك على Supabase:
   `https://nbkckaopgcopnpwkpfip.supabase.co`
2. افتح **SQL Editor → New query**.
3. الصق محتوى ملف `supabase-setup.sql` كاملاً واضغط **Run**.

هذا الملف ينشئ:
- جدول `admin_user` (حساب المطور) مع:
  - **اسم المستخدم:** `زياد`
  - **كلمة المرور:** `1963`
- جدول `works` (الأعمال)
- Bucket تخزين الصور `works` (عام)
- سياسات الأمان (RLS) اللازمة للاتصال بدون مشاكل

### 2) رفع الموقع
ارفع الملفات إلى GitHub ثم انشر عبر Vercel أو GitHub Pages.

### 3) لوحة المطور
من القائمة العلوية اضغط **✨ دخول سحري** ← أدخل اسم المستخدم `زياد` وكلمة المرور `1963` ← يمكنك:
- إضافة عمل (عنوان + وصف + صورة من الهاتف)
- تعديل أي عمل موجود
- حذف أي عمل

> لتغيير كلمة المرور لاحقاً: من Supabase → Table Editor → جدول `admin_user` → عدّل العمود `password`.
